# PYTHON IP CHECKER      

ip_checker is a python library which you can use to validate ip address and check whether an ip is private or public

* Usage:
 >>> from ip_checker import ip_checker
 >>> ip_obj = ip_checker('123.111.23.1')
 >>> ip_obj.is_valid()
 >>> True
 >>> ip_obj.is_public()
 >>> True
 >>> ip_obj.is_private()
 >>> False
